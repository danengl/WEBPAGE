HTML lists: Ordered and Unordered Lists

In this lesson, we’ll explore list elements in HTML, focusing on unordered lists (< ul >) and ordered lists (< ol >). Lists are common on websites—from articles like “Top 10 Must-Have Gadgets” to official lists like the “FBI’s Ten Most Wanted Fugitives.” Understanding how to create these lists will enhance your HTML skills.

  1. Unordered Lists (< ul >)

    An unordered list is used when the order of items doesn’t matter like a grocery list or a to-do list. These lists display items with bullet points.

    Basic Structure:

    < ul> < li >Milk< /li > < li >Bread< /li > < li >Eggs< /li > < /ul >

  • < ul >: Starts the unordered list (stands for unordered list).
  • < li >: Each item inside the list is wrapped in a list item tag (< li >).
  • < /ul >: Closes the unordered list.

Result:

  • Milk
  • Bread
  • Eggs
  1. Ordered Lists

    2. An ordered list is used when the sequence does matter—like instructions, rankings, or steps in a process. These items are numbered automatically.

    Basic Structure:

    < ol > < li >Wake up< /li > < li >Brush your teeth< /li > < li >Have breakfast< /li >

  • < ol >: Starts the ordered list (stands for ordered list).
  • < li >: Again, each item is inside a list item tag (< li >).
  • < /ol >: Closes the ordered list.

Result:

  1. Wake up
  2. Brush your teeth
  3. Have breakfast
